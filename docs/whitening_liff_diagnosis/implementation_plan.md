# ホワイトニングサロン「プラン診断」LIFFアプリ実装計画

このプロジェクトでは、ユーザーが4つの質問に答えることで、最適なホワイトニングプランを提案するLIFFアプリを開発します。

## ユーザーレビューが必要な項目
> [!IMPORTANT]
> - **LIFF ID**: ユーザーから提供された `2005378903-eFXXT5Gj` を使用します。
> - **GAS URL**: 現時点ではプレースホルダー（`https://script.google.com/macros/s/XXXXXXXX/exec`）を使用します。デプロイ後に差し替えが必要です。

## 検証プラン

### 自動テスト（ブラウザ操作）
- `npx vite` でローカルサーバーを起動し、ブラウザで診断フローが正常に動作することを確認します。
- LIFF SDKのモック処理を行い、実際のフローをシミュレートします。

### 手動確認
- スマートフォンの実機（LINEアプリ内）で以下の項目を確認：
  - 各質問が順番に表示されるか。
  - プログレスバーが正しく進むか。
  - 診断結果が仕様通りに判定されるか。
  - 「LINEに送る」ボタンでトーク画面に結果が送信されるか。
  - 送信完了後に完了画面が表示され、ウィンドウが閉じるか。
