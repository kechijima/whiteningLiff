# ホワイトニングサロン「プラン診断」完成報告

仕様書に基づき、単一の `index.html` ファイルで構成されたLIFF診断アプリを実装しました。

## 実装内容

### 1. 診断フロー
- **Intro画面**: アプリの目的を伝え、診断を開始します。
- **質問画面 (Q1-Q4)**: スムーズなアニメーションと共に質問を表示し、プログレスバーで進捗を可視化します。
- **分析中画面**: 分析時間を設けることで期待感を高めます。
- **結果画面**: 診断スコアに基づき、最適なプラン（4回/8回/12回）を提案します。

### 2. 機能
- **LIFF連携**: `liff.init` による初期化とプロファイル取得。
- **トーク送信**: `liff.sendMessages` を使用して、診断回答と結果をユーザーのトーク画面に投稿します。
- **データ保存**: 診断結果をGAS（Google Apps Script）のWebhookへPOST送信します。

### 3. デザイン
- 清潔感のあるブルー（`#6bc1ed`）を基調としたモダンなデザイン。
- スマートフォンに最適化されたUI部品（大きなボタン、読みやすいフォント）。

## ファイル構成
- `index.html`: 全てのHTML、CSS、JavaScriptが含まれる単一ファイル。

## 使用方法
1. LIFF ID: `2005378903-eFXXT5Gj` が設定されています。
2. GASのデプロイ後、`index.html` 内の `CONFIG.GAS_URL` を実際のエンドポイントに書き換えてください。
